Project Overview
This project predicts student academic performance using machine learning techniques.
It analyzes educational data such as study hours, attendance, previous scores, and other factors to predict student results.
The goal of this project is to use Educational Data Mining (EDM) techniques to identify patterns and help improve student performance.
